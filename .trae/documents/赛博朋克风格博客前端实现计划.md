# 赛博朋克风格博客前端实现计划

## 一、页面结构设计

### 1. 公共布局
- **顶部导航栏**：固定在顶部，包含博客标题和标签入口
- **响应式设计**：针对桌面端优化
- **背景效果**：深色渐变背景，带有轻微的网格或电路图案

### 2. 首页（文章列表）
- **英雄区域**：博客标题、副标题，带有霓虹发光效果
- **标签筛选**：横向标签栏，支持点击筛选
- **文章卡片列表**：网格布局，每张卡片包含标题、简介、发布时间、标签
- **加载状态**：科技感的loading动画

### 3. 文章详情页
- **文章头部**：标题、发布时间、标签，带有渐变效果
- **正文区域**：Markdown渲染，支持代码块高亮
- **阅读体验**：清晰的字体、合理的行距、科技感的代码块样式

### 4. 标签页
- **标签云**：展示所有标签，带有不同大小和颜色
- **文章列表**：点击标签后展示对应文章

## 二、组件拆分建议

### 1. 布局组件
- `Layout`：公共布局，包含导航栏和背景效果
- `NavHeader`：顶部导航栏组件

### 2. 首页组件
- `HeroSection`：英雄区域，展示博客标题
- `TagFilter`：标签筛选组件
- `ArticleCard`：文章卡片组件
- `LoadingSpinner`：科技感加载动画

### 3. 文章详情组件
- `ArticleHeader`：文章头部信息
- `MarkdownRenderer`：Markdown渲染组件

- `TagCloud`：标签云展示组件

## 三、赛博朋克风格设计

### 1. 配色方案
- **主色调**：深色背景（#0a0a0f）
- **霓虹色**：
  - 紫色（#9d4edd）
  - 蓝色（#4361ee）
  - 粉色（#f72585）
- **辅助色**：
  - 青色（#4cc9f0）
  - 绿色（#4895ef）

### 2. 字体选择
- **标题字体**：`Orbitron`（科技感无衬线字体）
- **正文字体**：`Inter`（清晰易读）
- **代码字体**：`JetBrains Mono`（等宽字体，适合代码展示）

### 3. 视觉效果
- **发光效果**：使用`text-shadow`和`box-shadow`实现霓虹发光
- **渐变效果**：线性渐变和径向渐变，用于背景和按钮
- **玻璃拟态**：半透明背景+模糊效果，用于卡片和导航栏
- **动画效果**：
  - 悬停时的霓虹高亮
  - 页面切换的淡入淡出
  - 标签选中的脉冲效果

### 4. 图标和装饰
- 使用科技感图标（如电路、芯片、霓虹灯效果）
- 背景添加细微的网格或电路图案
- 卡片边缘添加发光边框

## 五、示例代码

### 1. 首页示例代码
```tsx
// app/home/page.tsx
import { Article, Tag } from '@/types/article';
import Layout from '@/components/Layout';
import HeroSection from '@/components/HeroSection';
import TagFilter from '@/components/TagFilter';
import ArticleCard from '@/components/ArticleCard';
import LoadingSpinner from '@/components/LoadingSpinner';

async function fetchArticles() {
  const res = await fetch('/api/articles');
  return res.json();
}

async function fetchTags() {
  const res = await fetch('/api/tags');
  return res.json();
}

export default async function Home() {
  const [articles, tags] = await Promise.all([fetchArticles(), fetchTags()]);

  return (
    <Layout>
      <HeroSection />
      <TagFilter tags={tags} />
      <div className="container mx-auto px-4 py-8">
        <div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
          {articles.map((article: Article) => (
            <ArticleCard key={article.id} article={article} />
          ))}
        </div>
      </div>
    </Layout>
  );
}
```

### 2. 文章卡片组件示例
```tsx
// components/ArticleCard.tsx
import Link from 'next/link';
import { Article } from '@/types/article';

interface ArticleCardProps {
  article: Article;
}

export default function ArticleCard({ article }: ArticleCardProps) {
  return (
    <Link href={`/home/articles/${article.id}`} className="group">
      <div className="bg-gray-900/80 backdrop-blur-sm border border-purple-500/30 rounded-lg p-6 hover:border-purple-500 transition-all duration-300 hover:shadow-[0_0_15px_rgba(157,78,221,0.5)]">
        <div className="mb-4">
          {article.tags?.map((tag) => (
            <span key={tag.id} className="inline-block bg-purple-900/50 text-purple-300 text-xs px-2 py-1 rounded-full mr-2 mb-2">
              {tag.name}
            </span>
          ))}
        </div>
        <h3 className="text-xl font-bold text-white mb-2 group-hover:text-purple-400 transition-colors duration-300">
          {article.title}
        </h3>
        <p className="text-gray-400 mb-4 line-clamp-3">
          {/* 文章简介，这里可以从content中提取前150字 */}
          {article.content.slice(0, 150)}...
        </p>
        <div className="text-gray-500 text-sm">
          {new Date(article.created_at).toLocaleDateString()}
        </div>
      </div>
    </Link>
  );
}
```
